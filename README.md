A strategy game engine written in Java.

# Installation

## Dependencies

java 11

## From Source

```{{bash}}
./gradlew shadowJar
```

Produces a fat-jar in the path build/libs/sge-m.m.p-exe.jar

## Usage

```{{bash}}
java -jar sge-m.m.p-exe.jar
```
